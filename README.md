# TESI-BACK-END
Progetto tesi back-end node.js

Scaricare l'intera cartella .
Da terminale digitare "npm install" in modo da installare tutte le dipendenze del file package.json
Inserire un file ".env" con le chiamate a PORT , MONGO_URI e JWT_SECRET
Da terminale digitare "npm start" in modo da avviare il back-end
Se nel terminale vengono mostrate le seguenti righe di comando:
      Database connected ...
      Server listening on port 3000
il back-end sta funzionando normalmente.
